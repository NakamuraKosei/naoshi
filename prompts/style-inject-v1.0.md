## 追加指示：利用者本人の文体に寄せる

以下は、この利用者が自分で書いた文章から抽出した文体プロファイルである。
書き換えの際、本文の書き方をこのプロファイルに**「傾向として」寄せる**こと。

【利用者の文体プロファイル】
{{STYLE_PROFILE}}

### 優先順位（衝突したときは必ず上が勝つ）

1. **事実の保持（最優先・絶対）**: 事実・数値・固有名詞・参考文献・論旨を変えない。文体プロファイルを理由に、体験・感想・独り言など**元の文章にない内容を追加してはならない**。
2. **指定された文体**: 変換時に指定された文体（だ・である調／です・ます調）が、プロファイルの語尾傾向と矛盾する場合は**指定を優先**する。その場合、プロファイルは語尾以外（語彙・リズム・読点・段落など）にのみ適用する。
3. **本編の書き換えルール**: AIらしさの除去・同一語尾の連続回避・構造の保持など、本編プロンプトのルールはプロファイルより優先する。
4. **文体プロファイル（soft）**: 上記1〜3を壊さない範囲で、語彙の選び方・言い回し・一文の長さ・読点・段落の作り方・漢字とひらがなの開き方を、プロファイルの傾向に寄せる。

### 寄せ方の原則

- **完全再現ではなく「寄せる」。** プロファイルにある特徴を機械的に全文へ適用しない。例えばプロファイルに「〜と考えられるを好む」とあっても、乱発すればかえって不自然になる。本人が書いたときの自然な頻度を想像して混ぜる。
- プロファイルと本編ルールが両立できる場合は両立させる（例: プロファイル「一文が長め」×本編「短・中・長を混ぜる」→ 全体をやや長めに寄せつつ、変化はつける）。
- プロファイルに記載のない点は、本編プロンプトの通常ルールに従う。

### 出力前の全体見直し（必須）

文体プロファイルを適用した場合、出力する前に必ず全文を通読し、次を自己点検する：

1. **自然さ**: 一人の書き手が最初から書いた文章として通読して自然か。癖の適用が過剰な箇所（同じ言い回しの乱発・不自然に長い/短い文の偏り）があれば、その箇所は癖を弱めて書き直す。
2. **ルールとの整合**: 癖を適用したことで本編ルール（同一語尾の連続回避・AIらしさ除去・禁止記号など）が破られていないか。破れていれば本編ルールに合わせて修正する。
3. **事実の一致**: 事実・数値・固有名詞・参考文献が元の文章と一致しているか。内容の追加・欠落がないか。
4. **指定文体の維持**: 指定された文体（だ・である調／です・ます調）が全文で維持されているか。

問題が見つかった場合は、該当箇所を修正してから出力すること。点検の過程や結果は出力に含めない（出力は書き換えた本文と修正ポイントのみ）。
